# prompt-library 🚀

### AI, Content, and SEO toolkit for marketers.

A curated collection of production-ready prompts, structured data templates, and automation-friendly workflows designed to streamline digital marketing operations. This repository serves as a "digital filing cabinet" for scaling content strategy and technical SEO.

---

## 🛠️ What's Inside

This library is organized into functional modules to help marketers move from strategy to execution faster:

-   **Content Optimization:** Prompts for generating blog intros, meta descriptions, and PAS-framework copy with structured JSON outputs.
-   **Technical SEO:** Interactive schema generators (JSON-LD) for FAQs, How-To guides, and Article snippets.
-   **Workflow Automation:** Logic-heavy prompts designed for integration with tools like Zapier and GitHub.
-   **Generative Engine Optimization (GEO):** Evolving strategies for visibility in AI-driven search environments.

---

## 📋 Featured Prompts

### 1. Blog Intro & Meta Data Generator (JSON)
Generates SEO-optimized metadata and hooks.
- **Output:** Structured JSON
- **Frameworks:** Problem-Agitate-Solution (PAS)
- **Use Case:** CMS bulk uploads or automation pipelines.

### 2. Interactive Schema Generator
Converts raw content into valid JSON-LD.
- **Types:** FAQPage, HowTo, Article, VideoObject.
- **Benefit:** Increases SERP real estate and rich result eligibility.

---

## 🚀 How to Use

1. **Browse** the categories in the folder structure.
2. **Copy** the prompt text from the Markdown files.
3. **Paste** into your preferred LLM (GPT-4, Claude 3.5, or Gemini 1.5 Pro).
4. **Fill** in the bracketed variables like `[INSERT KEYWORD HERE]`.
